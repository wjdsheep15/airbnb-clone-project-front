# 🏡 Airbnb-clone-project-front
ASAC04 부트캠프 팀 프로젝트로 진행한 Airbnb 클론 프로젝트의 프론트엔드 레포지토리입니다.
Next.js 기반으로 실제 에어비앤비와 유사한 메인 페이지를 구현했습니다.

# ✨ 담당 역할
- 메인 페이지 구현
Airbnb UI를 참고하여 숙소 목록, 카드 구성, 반응형 등 메인 화면을 직접 구현했습니다.

- 동기화 작업 (Integration)
각 팀원들이 구현한 컴포넌트 및 기능들을 통합하고, 전체 흐름이 매끄럽게 연결되도록 라우팅 및 상태 관리, 레이아웃 조정 등을 담당했습니다.
이 과정에서 코드 충돌을 해결하고 일관된 코드 스타일을 유지하기 위해 노력했습니다.
